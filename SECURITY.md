# Security Policy

Nếu phát hiện lỗ hổng bảo mật, vui lòng không công khai ngay trên issue tracker.

Hãy tạo một kênh liên hệ riêng với người đang vận hành repo hiện tại hoặc mở private advisory trên GitHub nếu repo đã bật tính năng này.

Khi báo lỗi, nên cung cấp:

- mô tả vấn đề
- mức độ ảnh hưởng
- cách tái hiện
- đề xuất hướng giảm thiểu nếu có
