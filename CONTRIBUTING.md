# Contributing

Đây là dự án cộng đồng. Mọi đóng góp nên tập trung vào ba mục tiêu:

- giữ phạm vi thư viện gọn
- ưu tiên tính ổn định của flow đăng nhập và nhắn tin
- tránh thêm tính năng ngoài mục tiêu hiện tại nếu chưa có lý do rõ ràng

## Quy trình đề xuất

1. Tạo issue mô tả vấn đề hoặc đề xuất thay đổi.
2. Tách rõ thay đổi là sửa lỗi, cải thiện ổn định hay mở rộng API.
3. Gửi pull request nhỏ, có mô tả ngắn gọn và có thể build được.

## Quy ước

- Không thêm tính năng lớn ngoài phạm vi hiện tại nếu chưa thảo luận trước.
- Giữ API public đơn giản và tránh phát sinh phụ thuộc nặng.
- Mọi thay đổi cần chạy được `npm run lint` và `npm run build`.
